# Intoduction #

The project provide a platform independent calculator. Nothign super fancy

# Installation

```
pip install -i https://test.pypi.org/simple/ pancalc
```

# Usage

```
pancalc 3+4 # output 7
```

# For the developer

```
pip install -i https://test.pypi.org/simple/ pmake
pmake build install 
```
